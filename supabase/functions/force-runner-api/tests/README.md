# Pruebas de Force Runner API

Estas pruebas validan la capa de aplicacion sin conectarse a Supabase.

```powershell
deno test supabase/functions/force-runner-api/tests
```

Si Deno no esta instalado:

```powershell
irm https://deno.land/install.ps1 | iex
```

Luego reinicia la terminal y vuelve a ejecutar las pruebas.
