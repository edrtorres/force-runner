# Force Runner

Proyecto universitario de aplicacion Android para corredores, desarrollado con Java, Groovy, Supabase y arquitectura limpia.

## Documentacion principal

- Modelo de base de datos: `supabase/docs/database-model.md`
- APIs personalizadas: `supabase/docs/edge-functions-api.md`
- Reporte de pruebas de APIs: `supabase/docs/api-test-report.md`
- OpenAPI / Swagger: `supabase/docs/openapi.yaml`

## Swagger UI

La documentacion visual de APIs esta en:

```text
docs/swagger-ui/index.html
```

Cuando GitHub Pages este activo, se podra abrir con una URL similar a:

```text
https://USUARIO.github.io/REPOSITORIO/swagger-ui/
```

## Supabase

La configuracion de Supabase incluye:

- Migraciones de base de datos
- Edge Function `force-runner-api`
- Datos de prueba
- Documentacion OpenAPI
